## 1 优先处理的模式

| 模式 | 识别信号 | 处理方式 |
|---|---|---|
| 公式化开头 | “近年来受到广泛关注”、`In recent years`、`Despite recent advances` | 直接写问题、现有证据或具体缺口 |
| 空泛重要性 | “至关重要”“具有深远意义”、`pivotal`、`paramount` | 改为可观察后果、负担、机制或决策影响 |
| 宣传性创新 | “突破性”“首次”“革命性”、`groundbreaking`、`novel` | 只陈述与既往工作的可核实差异 |
| 空泛规模 | “大量”“全面”“广泛实验”、`extensive`、`comprehensive` | 写明样本、数据集、时间、指标或范围 |
| 模糊归因 | “已有研究表明”、`studies have shown` | 指向具体文献；无文献则降低为待核陈述 |
| 无证据强化 | “充分证明”“显著优于”、`prove`、`significantly outperform` | 对照研究设计、统计量、比较对象和证据位置 |
| 连接词堆叠 | 连续句以“此外/同时”、`Moreover/Furthermore` 起头 | 删除可由语义承担的连接词，必要时合并或重排 |
| 机械列举 | 为凑结构固定列三点或重复抽象贡献 | 保留真实数量，按证据重要性排序 |
| 同义词循环 | 同一构念被多个近义词轮换指代 | 固定术语，允许必要重复 |
| 句尾空转 | “凸显了重要性”、`highlighting the need` | 删除；如确有含义，改为具体结果或影响 |
| 复句堆叠 | 一个句子承担多个主张、多个从句或多层限定 | 按论断单元拆句，保持因果和限定关系 |
| 元话语 | “值得注意的是”“我们可以看到”、`It is worth noting` | 直接陈述事实 |

任何命中都先判断语境。合法术语、真实引文标题或准确描述不得为追求“归零”而改坏。

## 2 学术表达保留项

- 证据约束的审慎词：suggests、may、is consistent with、提示、可能、与……一致。
- 适当的被动语态和 `we`，不为追求表面变化而改写。
- 关键术语的重复。学术写作重在指代稳定，不做“优雅变化”。
- 方法名、模型名、量表名、公式、变量符号、引文键、表图编号和单位。
- 研究设计要求的限定条件，例如“在本队列中”“调整协变量后”“探索性分析”。
- 结果节必要的表图定位和讨论节必要的文献比较。

## 3 论断强度表

| 证据情形 | 可用表达 | 不应使用 |
|---|---|---|
| 描述性或横断面关联 | 相关、相伴、观察到、提示 | 导致、改善、保护、危害 |
| 观察性纵向结果 | 发生风险较高、与后续结局相关 | 证明因果、必然导致 |
| 随机试验 | 在研究条件和终点范围内降低或提高 | 普遍有效、永久改善 |
| 探索性或未校正分析 | 提示、方向一致、需验证 | 确立、证实、亚组获益 |
| 机制推测 | 可能与……有关、一种解释是 | 机制已明确、由……造成 |
| 基金申请预期 | 将检验、拟评估、预期产生 | 已证明、必将实现 |

## 4 逐部件审计重点

- 题名：研究设计与因果强度匹配；不靠“novel”抬高价值。
- 摘要：每个关键数值与正文和表格一致；结论不超出结果。
- 引言：缺口具体且由文献支持；研究目的逐项回应缺口。
- 方法：只写最终口径和可复现操作；不写分析日记或工程过程。
- 结果：数字与表图一致；不加入机制、评价或未预设结论。
- 讨论：每个主结果均有解释、文献对照和证据相称的意义；探索性结果不升格。
- 结论：不引入新结果，不把关联改写成因果，不重复整段摘要。

## 5 验证清单

- [ ] 改前与改后数字、单位、统计方向和精度完全一致。
- [ ] 引文数量、标识、顺序和归属未变；未新增虚构来源。
- [ ] 公式、符号、缩写、表图编号及交叉引用未变。
- [ ] 每个主要论断均可定位到数据、表图或引文。
- [ ] 动词强度与研究设计、统计证据和分析层级相称。
- [ ] 作者术语与声纹保持稳定，未被改成通用模板语气。
- [ ] 无口语、宣传腔、对话残留或程序实现痕迹。
